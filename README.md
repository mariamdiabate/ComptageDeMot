Projet Java : Comptage de mots dans un fichier texte

Ce projet Java lit un fichier texte et affiche le nombre total de mots qu'il contient.

 Fichier source

- Comptage.java : Le fichier source contenant le code Java pour lire un fichier et compter les mots.

Fichier texte

- FichierDeMot.txt : Fichier texte placé dans le même dossier que le programme. Ce fichier contient le texte à analyser.

 Fonctionnalités

- Lecture d'un fichier ligne par ligne
- Concaténation du contenu en une seule chaîne
- Découpage en mots (en se basant sur les espaces)
- Comptage du nombre total de mots
- Affichage du résultat

 Utilisation

1. *Compiler le programme* :

   ```bash
   javac Comptage.java

2. *Execter le programme* :
 
java Comptage
